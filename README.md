# M1 Tic-Tac-Toe

A simple implemntation of the imfamous **Tic-Tac-Toe** game with **React**.

![Tic Tac Toe Demo](demo/demo.gif)

# Hosted Demo

t3.avery.gg

# Installation

```bash

# Clone the repo and cd into the directory

# Install the depends.

yarn
# or
npm install

# Start the dev server

yarn dev
# or
npm run dev

```
